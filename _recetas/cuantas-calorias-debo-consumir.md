Tanto si nuestro objetivo es perder peso como si es ganarlo, necesitaremos saber cuántas calorías diarias deberemos consumir para conseguirlo. Y es que adelgazar o engordar dependen de una ecuación muy sencilla: adelgazamos si consumimos menos calorías de las que gastamos, y engordamos si consumimos más. Fácil, ¿verdad?

TASA METABÓLICA BASAL
Para calcular las calorías que necesitamos ingerir deberemos calcular primero nuestra Tasa Metabólica Basal (TMB), que serían las calorías que nuestro cuerpo consume en reposo, es decir, las que necesita para vivir. Esta tasa varía en función de las características de cada persona (peso, altura, edad…), y la fórmula para calcularla, también conocida como la ecuación de Harris-Benedict, es la siguiente:

Hombres: TMB = 66 + (13,7 x peso en kg) + (5 x altura en cm) – (6,8 x edad en años)
Mujeres: TMB = 655 + (9,6 x peso en kg) + (1,8 x altura en cm) – (4,7 x edad en años)
CALORÍAS DIARIAS QUE REALMENTE NECESITAMOS
Como la TMB indica las calorías que gastamos en reposo, deberemos multiplicarla por una cifra, según nuestra cantidad de ejercicio, para calcular las que necesitamos en realidad:

Poco o ningún ejercicio: TMB x 1,2
Ligero (1 a 3 días a la semana): TMB x 1,375
Moderado (3 a 5 días a la semana): TMB x 1,55
Fuerte (6 a 7 días a la semana): TMB x 1,725
Muy fuerte (entrenamientos duros dos veces al día): TMB x 1,9
Para que lo veáis con un ejemplo, las calorías que necesitaré consumir yo, que tengo 33 años, mido 1’63 m, peso 56 kg y realizo ejercicio 5 o 6 veces a la semana, serán las siguientes:

TMB = 655 + (9,6 x 56) + (1,8 x 163) – (4,7 x 33) = 655 + 537,60 + 293,40 – 155,10 = 1330,90

Calorías necesarias = 1330,90 x 1,55 = 2062,895

Estas son las calorías que necesito consumir para mantener mi peso actual. Sin embargo, para adelgazar necesitaré ingerir menos (entre 200 y 500 calorías menos para adelgazar de manera saludable) y para engordar necesitaré ingerir más.

En este sentido sí es importante tener una idea de las calorías que aportan los alimentos para poder hacer un seguimiento, ya que sin ese déficit o superávit calórico nuestro cuerpo no adelgazará ni engordará hagamos lo que hagamos.
